## 📌 Project Overview
This project analyzes social media engagement data across multiple platforms to uncover audience behavior, campaign effectiveness, sentiment patterns, and brand risk indicators. An interactive Power BI dashboard was built to transform raw data into actionable business insights.

---

## 🎯 Problem Statement
Brands and marketing teams often struggle to identify which social media content drives meaningful engagement across platforms. This project analyzes engagement trends, sentiment, emotions, and campaign performance to optimize content strategy and improve decision-making.

---

## 🛠️ Tools & Technologies
- **Power BI** – Dashboard development & data visualization
- **Python** – Data cleaning, EDA, NLP (Word Cloud, Hashtag Analysis)
- **Excel** – Data preprocessing
- **DAX** – KPI calculations and custom measures

---

## 📂 Dataset Features
The dataset includes:
- Platform data (YouTube, Facebook, Instagram, Twitter, Reddit, TikTok)
- Engagement metrics (Likes, Shares, Comments, Impressions)
- Sentiment & Emotion analysis
- Toxicity score
- Campaign details (Campaign Name, Campaign Phase)
- Hashtags and keywords

---

## 📊 Dashboard Highlights
✔ **598M+ Impressions** analyzed across platforms  
✔ **30M+ Engagements** tracked across campaigns  
✔ **27.81% Average Engagement Rate**  
✔ **72.97% Buzz Growth** detected  
✔ **50.39% Toxicity Score** monitored for brand risk  
✔ Highest engagement found during **Pre-Launch Campaign Phase (~50%)**

---

## 🔍 Key Insights
- **YouTube and Facebook** generated the highest engagement among all platforms.
- **Pre-launch campaigns** outperformed launch and post-launch phases.
- **Likes (62.23%)** dominated interactions, followed by shares and comments.
- Balanced sentiment highlighted diverse audience opinions.
- Toxicity analysis emphasized the importance of monitoring negative content.
